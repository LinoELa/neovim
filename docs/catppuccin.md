# catppuccin

<!-- Config: lua/plugins/catppuccin.lua -->

Tema de color (claro/oscuro automático).

- Repo: https://github.com/catppuccin/nvim
- `flavour = "auto"` → latte (claro) / mocha (oscuro)

## Cambios propios

- `no_italic = true` en resaltado general (comentarios siguen en cursiva).
- Integraciones: cmp, gitsigns, mini.

## Comando

```vim
:colorscheme catppuccin
```
