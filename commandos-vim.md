# Comandos nuevos

Nota: en esta configuracion `<leader>` es `Espacio`.

Solo anoto aqui los atajos nuevos que vamos agregando, no todos los de Neovim o LazyVim.

## Snacks

- `Espacio e`: abre el explorador de archivos (`Snacks.explorer()`).
- `Espacio Espacio`: busca archivos con el picker inteligente (`Snacks.picker.smart()`).
- `Espacio ,`: muestra buffers abiertos (`Snacks.picker.buffers()`).
- `Espacio /`: busca texto en el proyecto (`Snacks.picker.grep()`).
- `Espacio :`: abre el historial de comandos (`Snacks.picker.command_history()`).
- `Espacio n`: abre el historial de notificaciones (`Snacks.picker.notifications()`).

  │ Espacio /       │ Buscar texto en el proyecto │
